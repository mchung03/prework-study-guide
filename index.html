HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

    • Cheat Key

  • p = paragraph or block of text
  • u1 = unordered lists
  • o1 = ordred lists
  • li = list items
  • img = images, src = location of image
  • br = line of empty space, or line break
  • a = anchor, creates links to same webpage

CSS

  • A margin indicats how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS: applies to specific HTML tag
  • Internal CSS: CSS rules for webpage in header
  • External CSS: keeps CSS rules in separate file, makes design changes easier (most common)

Git

  • git status: checks what branch we are currently on.
  • git checkout main: takes you to main branch.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git branch -m (wrong name) (right name): renames branch.

    • Git Flow

  • git checkout branch-name: go to branch
  • git add -A: adds changes
  • git commit -m "name of change": commits to repository
  • git pull origin main: updates branch
  • git push origin branch-name: pushes changes
  • git checkout main: updates main branch
  • git pull oriin main: pulls remote main branch into local main branch
